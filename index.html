MICROMACHINES

Requirements:

You have a micromachine car, you place it on a 10x10 track at position [0,0] facing North. Given a set of instructions, the micromachine should navigate the track without 'falling off'. Satisfy the following instruction sets:

  • [ 3R3R3 ] position: [0,3]S
  • [ R3L1L2] position: [1,1]W
  • [ 9R9R2 ] position: 'Off the Track!'
